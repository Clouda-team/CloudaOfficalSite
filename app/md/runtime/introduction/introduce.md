# Runtime简介

## 概述

### Runtime

Runtime为轻应用在移动终端上可以安全、稳定、高效的运行提供运行环境，在该运行环境中提供了一套方便快捷的插件机制，可以快速地让轻应用获得与Native应用一样的能力，并提供权限管理，生命周期管理等APP管理功能。
