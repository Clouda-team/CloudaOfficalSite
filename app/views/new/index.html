Clouda+组成部分
  • Blend Blend是一套JavaScript的API，将Native的端能力和百度的云服务融合（Blend）到webapp中。
    Blend 文档 实例展示
  • Runtime Runtime在移动终端上为webapp提供安全、稳定、高效的运行环境并且提供了一套灵活的插件机制，让webapp具有和本地应用一样的能力。
    Runtime 文档 打包平台
  • Rapid JS 基于Nodejs的服务端开发框架，为移动而生，让移动web开发简单、灵活、扩展能力强。
